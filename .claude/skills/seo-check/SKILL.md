# SEO Audit

## For each page, check:

### Technical SEO
- [ ] Title tag: 50-60 chars, keyword + "California"
- [ ] Meta description: 150-160 chars, keyword + CTA
- [ ] Canonical URL present and correct
- [ ] One H1 with primary keyword
- [ ] Heading hierarchy (no H3 without parent H2)
- [ ] All images: alt text, next/image, width/height
- [ ] No broken internal links
- [ ] Breadcrumbs present

### Structured Data
- [ ] LocalBusiness JSON-LD
- [ ] Service/FAQPage/BlogPosting schema where needed
- [ ] BreadcrumbList schema
Validate: https://validator.schema.org

### Content SEO
- [ ] Primary keyword in H1, first paragraph, meta description
- [ ] 2+ internal links to other pages
- [ ] FAQ section with 4+ questions

## Output: scorecard table with ✅/❌ and overall score X/100
